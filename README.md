# Finnish name generator

Generates random Finnish names.

## Requirements

Node, npm and grunt.

## Usage

Install dependencies

```
$ npm install
```

Prepare json files from name lists

```
$ grunt
```

Generate random names

```
$ node app.js 10
Bia Suvela
Mats Nissi
Greger Sotavalta
Hilel Pensakkala
Maria-Hanna Katajamäki
Jasu Turska
Igor Arosuo
Tuike Haukka
Jon Suuti
Anni-Maria Pitkämäki
```
