# Finnish name generator

## Requirements

Node and NPM.

## Usage

Install local grunt

```
$ npm install
```

Generate json files from name lists

```
$ grunt
```

Generate random names

```
$ node app.js 10
Bia Suvela
Mats Nissi
Greger Sotavalta
Hilel Pensakkala
Maria-Hanna Katajamäki
Jasu Turska
Igor Arosuo
Tuike Haukka
Jon Suuti
Anni-Maria Pitkämäki
```
